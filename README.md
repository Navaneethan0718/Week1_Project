# Sales Data Analysis Using NumPy

## Objective
To analyze supermarket sales data stored in CSV format using NumPy array operations
and statistical functions.

## Dataset
The dataset includes:
- Product Category
- Quantity Sold
- Unit Price
- Sales Amount

## Tools Used
- Python
- NumPy
- Google Colab

## Conclusion
This project demonstrates efficient data analysis using NumPy vectorized operations
on real-world sales data.
